# 다시나 (Dasina) — 뇌졸중 후 통합 회복 플랫폼

뇌졸중 후 마비말장애(dysarthria) 환자를 위한 AI 기반 재활 솔루션

---

## 앱 구성

| 앱 | 역할 | 핵심 기능 |
|----|------|----------|
| **말벗** | 언어 재활 | 음성 분석 → 중증도 판정 → 16가지 맞춤 운동 처방 |
| **마음벗** | 우울증 치료 | rPPG 비접촉 심박 측정 → 우울도 판정 → 훈련 강도 자동 조절 |

## 핵심 설계 원칙

1. **비판 금지** — 실패해도 부정적 피드백 없음
2. **속도보다 반복** — 매일 하는 것이 핵심, 포기 방지 시스템
3. **감정 케어 필수** — 우울도에 따라 훈련량 자동 조절

## 2축 독립 조절 시스템

| 축 | 입력 | 조절 대상 |
|----|------|----------|
| 기분 (rPPG) | HRV 심박변동성 | **몇 가지** 할지 (3~9개) |
| 난이도 (Level) | 운동 결과 누적 | **얼마나 어렵게** 할지 (Lv.1~5) |

- 우울 높은 날: 3가지만, 하지만 레벨 목표는 유지
- 우울 낮은 날: 9가지, 같은 레벨 목표
- 3회 연속 목표 달성 → 자동 레벨업
- 3회 연속 목표 실패 → 자동 레벨다운 + "천천히 다시 쌓아가요"

## 4대 치료 영역 × 16가지 운동

| 영역 | 운동 |
|------|------|
| 호흡 | 자세교정, 호흡운동, 하품-한숨 |
| 발성 | 지속발성, 글라이딩↑↓, 피치조절, 볼륨업, 성문폐쇄 |
| 조음 | 구강운동, 대조훈련(최소대립쌍), DDK 직접훈련 |
| 운율 | 대립강세, 탭핑, 천천히읽기, 크게읽기 |

## 평가 체계

| 평가 과제 | 측정 영역 |
|----------|----------|
| 아, 이, 우 모음발성 | 호흡 + 발성 (MPT, 음질, F0, 음량) |
| 퍼터커 DDK | 조음 (음절 반복 속도, 정확도) |
| 단어읽기 30개 | 조음 + 운율 (명료도, 말속도) |

**재평가 주기:** 급성기 2주 / 아급성기 4주 / 만성기 8주 / 유지기 12주

## 사용자 플로우

```
초기 설정 → 홈 → rPPG 측정 → 훈련 목록 → 운동 실행 → 세션 완료
  (성함,       (모드 선택)  (우울도 판정)  (과제 조합)   (Phase A→B/C)  (결과+레벨+
   생년,                                               자동측정/      스탬프+
   발병일)                                              자가확인)      가족 공유)
```

## 골든타임 D-day 시스템

발병 후 180일 이내 집중 재활이 회복률을 결정 (신경가소성 이론).

| 병기 | 경과일 | D-day 배지 | 권장 훈련 강도 |
|------|--------|-----------|-------------|
| 급성기 | 0~30일 | 빨강 | 매일 30~40분 |
| 아급성기 | 31~90일 | 주황 | 매일 40~50분 |
| 만성기 | 91~180일 | 회색 | 주 5회 30~40분 |
| 유지기 | 180일+ | 회색 | 주 3회 이상 |

### 프로토타입 컨트롤 (상단 검정 바)

| 컨트롤 | 설명 |
|--------|------|
| 우울(rPPG) | 높음/보통/낮음 → 과제 수 변경 |
| 중증도 | 중도/중등도/경도 → 과제 종류 변경 |
| 모드 | 통합/집중 → 재활 모드 전환 |
| 레벨 | Lv.1~4 → 전체 운동 레벨 변경 |
| 화면 | 각 화면 직접 이동 |

## 이론적 근거

| 이론 | 적용 |
|------|------|
| Motor Learning Theory | 반복 + 피드백 → 자동화, 80% 규칙 레벨 시스템 |
| VFE (Stemple 1994) | 성대 기능 강화 단계적 훈련 |
| LSVT LOUD (Ramig 1995) | 음량 증대 → 발성 시스템 활성화 |
| 행동활성화이론 | 작은 성공 → 보상 → 우울 완화 |
| 인지부하이론 | 우울 = 인지자원↓ → 과제 단순화 |
| 자기효능감 (Bandura) | 성공 경험 → 지속성↑ |
| 신경가소성 이론 | 반복 + 지속 = 뇌 재학습 |
| Yerkes-Dodson 법칙 | 현재 각성 수준에 맞는 난이도 = 최적 |
| DTx 설계 원칙 | Core Loop 유지 + 최소단위 + 정서결합 |
